# bartoszgrabski.dev — web app

Next.js web application for Bartosz Grabski's personal portfolio.

## Development

```bash
npm run dev:web        # from monorepo root
# or
npm run dev            # from apps/web
```

## Testing

```bash
npm test               # run Jest tests
npm run test:watch     # watch mode
```

## Build

```bash
npm run build:web      # from monorepo root (static export to apps/web/out/)
```

See root README and `DEPLOYMENT.md` for deployment instructions.
